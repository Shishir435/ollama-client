---
title: Keyboard Shortcuts
description: Default keyboard shortcuts for Ollama Client.
---

All shortcuts are customizable in Settings → Shortcuts. Defaults are listed below.

| Action | Default Key | Description |
|---|---|---|
| Focus Input | `/` | Focus the chat input field |
| Toggle Sidebar | `Alt+B` | Open or close the sidebar |
| Open Settings | `Alt+O` | Open the settings page |
| Search Messages | `Alt+F` | Open semantic chat search |
| New Chat | `Alt+N` | Create a new chat session |
| Stop Generation | `Escape` | Stop the AI response |
| Clear Chat | `Alt+X` | Clear the current chat session |
| Copy Last Response | `Ctrl+Shift+C` / `Cmd+Shift+C` | Copy the last AI response to clipboard |
| Export as JSON | `Alt+J` | Export current chat as JSON file |
| Export as Markdown | `Alt+M` | Export current chat as Markdown file |
| Export as PDF | `Alt+P` | Export current chat as PDF file |
| Export as Text | `Alt+Shift+T` | Export current chat as plain text file |
| Toggle Theme | `Alt+T` | Switch between light and dark mode |
| Toggle RAG | `Alt+R` | Enable or disable context retrieval |
| Toggle Speech | `Alt+S` | Start or stop speaking the last AI response |
| Toggle Tabs | `Alt+K` | Enable or disable tab access |
| Toggle Session Metrics | `Alt+M` | Show or hide the session metrics bar |

## Categories

**Navigation** — move around the interface: Focus Input, Toggle Sidebar, Open Settings, Search Messages.

**Actions** — perform common tasks: New Chat, Stop Generation, Clear Chat, Copy Last Response, and all export actions.

**Toggles** — switch features on/off: Toggle Theme, Toggle RAG, Toggle Speech, Toggle Tabs, Toggle Session Metrics.

## Notes

- `Mod` refers to `Ctrl` on Windows/Linux and `Cmd` on macOS.
- Shortcuts are persisted via `chrome.storage.sync` and synced across devices.
- Conflicting shortcuts are flagged in the settings UI and must be resolved before saving.
