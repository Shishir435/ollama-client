## Reporting a Vulnerability

We take the security of this project seriously. If you discover a vulnerability, please report it so we can address it as soon as possible.

### How to Report

Please do **NOT** report security vulnerabilities via public GitHub issues.

Instead, please report them via email to **shishirchaurasiya435@gmail.com**.

Please include:
*   A description of the vulnerability.
*   Steps to reproduce the issue.
*   Any relevant screenshots or code snippets.

### Process

1.  We will acknowledge receipt of your report within 48 hours.
2.  We will investigate the issue and determine its impact.
3.  We will keep you updated on our progress.
4.  Once fixed, we will release a security update.
